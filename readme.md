# вҡҷпёҸ Installation

## 1. Clone Repository

```bash
git clone https://github.com/lakshyawardhansinghrathore/MoodShift
cd MoodShift
```

## 2. Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

## 3. Install Dependencies

